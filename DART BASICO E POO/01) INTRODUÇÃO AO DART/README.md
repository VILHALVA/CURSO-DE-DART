# INTRODUÇÃO AO DART
Dart é uma linguagem de programação desenvolvida pela Google, lançada inicialmente em 2011. É uma linguagem de alto nível, orientada a objetos, com forte tipagem estática opcional. Ela é conhecida por ser utilizada principalmente para o desenvolvimento de aplicativos móveis, web e servidor.

## Características Principais de Dart:
1. **Orientada a Objetos:**
   - Dart é uma linguagem orientada a objetos, onde tudo é um objeto. Ela suporta os conceitos fundamentais da POO, como classes, objetos, herança, polimorfismo e encapsulamento.

2. **Tipagem Estática Opcional:**
   - Dart permite tanto tipagem estática quanto dinâmica, tornando-a flexível para diferentes estilos de desenvolvimento. Você pode optar por especificar tipos para variáveis e parâmetros de função, ou deixar o sistema de tipos inferir automaticamente.

3. **Eficiência e Desempenho:**
   - Dart é projetada para ser rápida e eficiente. Ela utiliza um mecanismo de execução otimizado (como o Dart VM ou compiladores para JavaScript) para alcançar desempenho próximo de linguagens nativas.

4. **Suporte a Programação Assíncrona:**
   - Dart possui suporte nativo para programação assíncrona, facilitando o desenvolvimento de aplicativos que requerem operações não bloqueantes, como chamadas de API e E/S de arquivos.

5. **Ferramentas e Ecossistema:**
   - Dart oferece um conjunto robusto de ferramentas, incluindo um compilador (Dart VM), um compilador para JavaScript (dart2js), e o Flutter SDK para desenvolvimento de aplicativos móveis multiplataforma.

6. **Multiplataforma:**
   - Uma das grandes vantagens de Dart é sua capacidade de compilar para várias plataformas. Ela pode ser usada para desenvolver aplicativos para Android, iOS, web e também para o desenvolvimento de backends.

7. **Flutter:**
   - Dart é especialmente conhecida por ser a linguagem de programação usada com o framework Flutter, também desenvolvido pela Google. Flutter é um framework UI para criar aplicativos nativos compilados a partir de um único código base.

## Exemplo Básico em Dart:
Aqui está um exemplo simples de um programa Dart que imprime "Olá, mundo!":

```dart
void main() {
  print('Olá, mundo!');
}
```

Neste exemplo:
- `void main()` é a função principal que marca o ponto de entrada do programa.
- `print('Olá, mundo!')` é uma função que exibe uma mensagem na saída padrão.

## Ambientes de Desenvolvimento Dart:
Para desenvolver em Dart, você pode usar diversas ferramentas e ambientes:

- **DartPad:** Um editor online que permite escrever e executar código Dart diretamente no navegador.
- **IDEs:** Como Visual Studio Code com extensões Dart e Flutter, IntelliJ IDEA com plugin Dart, entre outros.
- **Terminal/Console:** Para executar programas Dart diretamente a partir da linha de comando.

Dart é uma linguagem versátil que combina eficiência, simplicidade e poder, adequada para uma variedade de aplicações desde desenvolvimento móvel até sistemas complexos de backend.
